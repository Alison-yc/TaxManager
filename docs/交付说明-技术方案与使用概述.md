**文档版本**：v0.3  
**文档日期**：以项目更新为准  
**文档说明**：本文面向业务与技术对接方，用于说明系统定位、约定范围、使用方式及后续扩展方向；内容随项目迭代更新。

## 一、项目定位与能带来的价值

本项目面向**少量内部岗位**使用，在**不建设独立服务器、不雇佣专职运维**的前提下，实现：

- **账号登录**：用户与数据托管在成熟云服务上。
- **数据集中管理**：业务数据存于云数据库，减少分散在本地文件中的情况。
- **固定操作路径**：以**官方/约定版式 Excel 为唯一数据入口**（导入），在工作流中完成 **列表查阅 → 预览 → 导出存档（PDF）**，避免随意改数、误删等操作风险。
- **可演进**：在现有架构上，可按管理要求继续增加**版式级导出、字段细化、权限分层**等。

> 当前阶段明确：**不提供**对单条记录在界面上的随意「编辑」「删除」（即不包含常见后台里的 U/D）；数据以导入为准进入系统，导出以预览页为准存档。

## 二、技术选型说明（为何这样选）

| 层级 | 选型 | 业务侧可感知的含义 |
|------|------|--------------------|
| 前端 | **React 18** + **Vite** + **路由** | Web 浏览器访问；可区分「列表 / 预览」等页面步骤。 |
| 后端/账号/数据库 | **Supabase**（认证 + PostgreSQL） | **无需自建服务器**；权限与策略在云端统一配置。 |
| 数据来源 | **固定 Excel** | 模版版式相对稳定，导入规则可维护、可减少「随便一张表」。 |
| 发布 | **静态站点托管** + **自动化构建发布** | 访问地址固定、升级可追踪。 |

**小结**：**Web + 云数据库**。当前阶段核心是 **Excel 导入 → 可查列表 → 预览 → PDF 存档**，与设计目标一致。

## 四、操作流程与菜单约定（与对方需求对齐）

### 4.1 必选主路径（本期启用）

1. **登录**  
2. **导入 Excel**（固定模版；由需求方提供并锁定版式）  
3. **数据列表**：展示一条**可检索摘要**列，并支持按摘要/编号/内容关键词过滤  
4. **进入预览**：从列表进入**单条记录预览页**（只读）  
5. **导出 PDF**：在预览页生成/保存 **PDF**（实现上可与浏览器「打印为 PDF」能力配合，具体以验收约定为准）  

### 4.2 本期「不做 / 伪装」的入口

- **不提供**面向业务人员的**逐条编辑、逐条删除**（即不做传统 CRUD 里的 U、D）。  
- 侧栏中除「数据列表」外，其它导航项本期为**置灰占位**（不可用），以保持界面结构与未来扩展余地，不构成承诺功能。

## 五、数据与 Excel / PDF

### 5.1 业务表（概念）

仍以 **`form_data`** 等为载体；每次成功导入会形成一条可追溯记录。**内容**中包含解析后的Excel数据及用于列表展示的**摘要字段**，便于检索与对接后续「按模版回填式导出」。

### 5.2 Excel 与 PDF 约定（与需求对齐）

| 环节 | 说明 |
|------|------|
| **导入** | 使用**锁定版式的 Excel 模版**；用户填好后上传导入；模版变更需同步更新解析规则。 |
| **列表检索** | 至少提供一列**可读摘要**（如由纳税人名称、税号等关键单元格推导），并支持关键字筛选。 |
| **预览** | 只读展示；当前实现阶段可先以「结构化预览 + JSON 存档」过渡到「与模版版式一致的版式预览」。 |
| **导出 PDF** | 在预览步骤输出 **PDF**：版式上以与「同模版」一致为目标；是否与本地 Excel 「另存为 PDF」完全一致，以**阶段性验收口径**为准。 |

## 六、权限与账号（现状与可扩展）

- **现状**：能登录的成员，按当期策略可进行**导入、查阅、预览与导出路径**上的操作（以线上为准）。  
- **可扩展**：按岗位限制「仅可查」「可导入」「可导出」等，在技术方案上与数据库策略结合实现（后续单独立项）。

## 七、使用说明（写给业务操作人员）

1. 使用分配的账号浏览器登录系统。  
2. 在 **「数据列表」** 页使用 **「导入 Excel」** 上传符合模版的文件。  
3. 在列表中通过 **「可检索摘要」** 或检索框找到目标记录。  
4. 点击 **「预览与导出」** 进入预览页，确认内容后使用 **「导出 PDF」** 保存为 PDF 文件。  
5. 侧栏其他菜单当前为**未开放**，请勿作为操作依据。
